Wiring rules (keep simple)
IDs: treat all *Id as snowflake. snowflake ↔ string allowed.
Channels: when a node expects TextBasedChannel, any text channel subtype is valid; for IDs use channelId (snowflake).
Objects → scalars: disallow direct object → string. Use extractor nodes (Message → content, Channel → channelId).
Numbers: int/float auto-fit into number. Avoid number → bigint.
